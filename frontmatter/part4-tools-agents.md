Agents are Claude using tools in a loop to accomplish a goal. This part opens
with Anthropic's engineering essays on building effective agents, engineering
their context, writing good tools, and designing harnesses for long-running
work — the principles worth internalizing before you write code. It then turns
to the platform's tool-use reference, the *Tool Use* course, and cookbook
patterns for workflows, orchestration, the Agent SDK, and managed agents.
