Claude Code is Anthropic's agentic coding tool for the terminal, IDE, desktop,
and web. This part begins with the best-practices essay, then works through the
documentation: how it works, everyday workflows, memory and settings, slash
commands, subagents, hooks, plugins, MCP, permissions and sandboxing, headless
and CI usage — and closes with the Claude Agent SDK for building your own
production agents on the same foundation.
