# Preface

This book is a curated, offline-readable compilation of Anthropic's official
guidance for building with Claude. It gathers material from four kinds of
first-party sources and weaves them into a single reading order:

- **Developer Platform documentation** (`platform.claude.com`) — how the API,
  tools, and capabilities actually work.
- **Claude Code documentation** (`code.claude.com`) — the agentic coding tool,
  its configuration surface, and the Agent SDK.
- **Cookbooks, courses, and the interactive prompting tutorial** — the hands-on,
  notebook-driven teaching material from Anthropic's public repositories.
- **Engineering essays** — the field-tested principles behind effective agents,
  tools, context engineering, and skills.

It is organized into eight parts that move from first principles to production:
Foundations, Prompt Engineering, Building with Claude, Tools & Agents, Agent
Skills, the Model Context Protocol, Claude Code, and a closing collection of
Cookbook recipes. Each chapter carries a *Source:* line pointing back to the
original file or page, so you can always find the canonical, always-current
version online.

Everything here is published, official Anthropic material, compiled for personal
offline study. Each source remains under its original license and copyright
(© Anthropic and the respective repository licenses). When in doubt, the live
versions at `docs.claude.com`, `code.claude.com`, and `github.com/anthropics`
are authoritative.
